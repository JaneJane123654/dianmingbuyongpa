# 学生课堂挂机程序 - 技术实现文档 v3.0 (Java版)

## 一、项目描述

### 1.1 项目背景
本程序是一款面向学生的课堂辅助工具，通过语音识别、唤醒词检测和AI智能问答技术，帮助学生在课堂环境中及时响应老师提问，避免因走神而错过重要信息。

### 1.2 核心价值
- **智能唤醒**：通过自定义名字作为唤醒词，实时监测是否被点名
- **安静检测**：使用Sherpa-ONNX VAD模型检测语音活动，智能区分语音和环境噪音
- **智能问答**：自动记录课堂内容并生成AI回答，帮助学生快速准备答案
- **隐私保护**：数据本地存储，不上传至云端

### 1.3 目标平台
- **主要平台**：Android
- **后期规划**：扩展支持 Windows、macOS
- **实现方式**：Java 全栈架构（JavaFX UI + Sherpa-ONNX JNI + LangChain4j），一套代码优先适配 Android，后期可扩展至其他平台
- **语言统一**：全部使用 Java 实现（包括 UI 层 JavaFX 和业务逻辑层）

## 二、技术选型

### 2.1 核心技术栈

#### 2.1.1 UI框架
- **JavaFX**
  - **选型理由**：
    - 跨平台 UI 框架，支持 Android、Windows、macOS
    - 现代化声明式 UI（FXML + CSS），开发体验优秀
    - 性能优异，硬件加速渲染
    - Java 生态成熟，社区活跃
    - Android 原生支持（通过 Gluon 或 JavaFXPorts）
    - Windows/macOS 原生支持
  - **技术特点**：
    - FXML：声明式 UI 描述语言
    - JavaFX Media：跨平台音频处理
    - Java HTTP Client：HTTP 客户端
    - 丰富的 UI 组件库（ControlsFX）

#### 2.1.2 语音识别与唤醒词检测
- **Sherpa-ONNX (JNI)**
  - **选型理由**：
    - 轻量级推理引擎，仅3.3M模型大小
    - **支持自定义唤醒词，无需重新训练**（关键优势）
    - 支持实时音频流处理
    - 支持多平台（Android、Windows、macOS）
    - 通过 JNI（Java Native Interface）调用 C++ 库
    - 开源免费（Apache 2.0协议）
  - **JNI 集成方式**：
    - **Android 平台**：使用 Sherpa-ONNX 官方 Android SDK（Kotlin/Java）
    - **Windows/macOS 平台**：编写 JNI 桥接代码，调用 Sherpa-ONNX C++ 库
  - **推荐模型**：
    - 唤醒词：`sherpa-onnx-kws-zipformer-wenetspeech-3.3M-2024-01-01`
    - 语音识别：`sherpa-onnx-zh-wenetspeech-zh`
    - VAD模型：`sherpa-onnx-vad-silero-vad`（安静检测使用）
  - **技术特点**：
    - 基于Zipformer架构的轻量级关键词检测模型
    - 唤醒词设置：支持拼音建模，可直接配置任意中文唤醒词
    - VAD检测：基于Silero的语音活动检测模型
    - **无需训练，仅需修改配置文件**

#### 2.1.3 音频录制
- **平台适配音频录制**
  - **选型理由**：
    - 根据平台使用原生音频 API，保证最佳性能和兼容性
    - 统一的 Java 音频数据接口，便于后续处理
  - **Android 平台**：
    - 使用 `MediaRecorder` 或 `AudioRecord` API
    - 原生支持，性能优异
  - **Windows/macOS 平台**：
    - 使用 `javax.sound.sampled` (Java Sound API)
    - 跨平台一致 API
  - **技术参数**：
    - 采样率：16kHz（语音识别标准）
    - 通道数：单声道（MONO）
    - 音频格式：PCM 16-bit
    - 编码格式：WAV（最通用，兼容性好）
    - 缓冲区大小：根据采样率动态计算（建议4096字节）

#### 2.1.4 安静检测（VAD）
- **Sherpa-ONNX VAD模型（Silero VAD，JNI 调用）**
  - **选型理由**：
    - 基于深度学习的语音活动检测（Silero模型）
    - 比RMS音量检测更智能，能准确区分"语音"和"噪音"
    - 过滤翻书声、椅子移动等环境噪音
    - 与唤醒词检测同一生态，统一使用Sherpa-ONNX
    - 通过 JNI 调用 C++ 库
  - **技术特点**：
    - 模型大小：约60MB
    - 准确率：高（相比RMS）
    - 延迟：30-50ms
    - 支持：跨平台（Android、Windows、macOS）

#### 2.1.5 AI大模型集成
- **LangChain4j（开源大模型调用框架）**
  - **选型理由**：
    - 纯 Java 实现，无平台依赖
    - 一键调用多种大模型 API（OpenAI、百度千帆、阿里通义千问、腾讯混元等）
    - API 统一，易于切换不同模型
    - 流式响应支持
    - 对话记忆管理
    - 简单易用的 API
  - **支持的大平台**：
    - OpenAI（GPT-4、GPT-3.5）
    - 百度千帆（ERNIE-Bot 系列）
    - 阿里云通义千问
    - 腾讯混元
    - 本地大模型（Ollama）
  - **集成方式**：Java SDK 调用
    - **认证方式**：直接使用 API Key / Access Token
    - 推荐模型：百度千帆 ERNIE-Bot-turbo（响应速度快，适合实时问答）
    - 响应格式：流式 JSON / 普通 JSON
  - **代码示例**：
    ```java
    // 一键切换不同模型
    ChatLanguageModel model = OpenAiChatModel.builder()
        .apiKey("your-key")
        .modelName("gpt-4")
        .build();

    // 或使用百度千帆
    ChatLanguageModel model = QianfanChatModel.builder()
        .apiKey("your-key")
        .secretKey("your-secret")
        .modelName("ERNIE-Bot-turbo")
        .build();

    // 统一调用接口
    String response = model.generate("你好");
    ```

#### 2.1.6 跨平台架构
- **Java 全栈架构（Android + Windows + macOS）**：
  - **UI层**：JavaFX（跨平台统一 UI）
  - **业务逻辑层**：Java（跨平台共享）
  - **音频层**：平台适配 API（Android: MediaRecorder/AudioRecord, Desktop: Java Sound）
  - **推理层**：Sherpa-ONNX（C++ 库，通过 JNI 调用）
  - **数据层**：Java Preferences / Java Properties（跨平台配置管理）

### 3.2 详细功能说明

#### 3.2.1 功能1：实时录音
**功能描述**：点击"上课"按钮后，应用开始实时录音，并将音频数据存入环形缓冲区，用于后续唤醒词检测和语音识别。

**实现要点**：
- 使用平台适配的音频 API 进行实时音频采集
  - Android: `MediaRecorder` 或 `AudioRecord`
  - Windows/macOS: `javax.sound.sampled` (Java Sound API)
- 采样率：16kHz，单声道，16-bit PCM
- 音频格式：WAV（便于后续处理）
- 录音数据存入环形缓冲区（Circular Buffer）
- 使用原子操作实现线程安全（读写分离，无写冲突）
- 支持暂停/继续/停止操作

**用户交互流程**：
```
1. 用户点击"上课"按钮
2. 应用请求录音权限
3. 开始实时录音，界面显示录音状态
4. 后台同时运行唤醒词检测和安静检测
5. 用户点击"下课"按钮停止录音
```

#### 3.2.2 功能2：唤醒词配置
**功能描述**：用户输入自己的名字（如"张三"），系统使用Sherpa-ONNX的预训练模型配置该名字作为唤醒词。

**技术实现**：
- Sherpa-ONNX使用拼音建模单元（声母+韵母）
- 无需重新训练，通过配置文件设置唤醒词
- 支持多个唤醒词（如姓名的多个发音变体）
- 动态生成keywords.txt配置文件

**配置示例**：
```json
{
  "keywords": [
    "张三",
    "张三同学"
  ],
  "threshold": 0.5
}
```

**用户交互流程**：
```
1. 用户进入设置界面
2. 输入姓名（支持多个变体，逗号分隔）
3. 点击"保存"配置唤醒词
4. 系统验证唤醒词有效性
5. 生成keywords.txt文件
6. 保存配置并提示成功
```

#### 3.2.3 功能3：唤醒词检测
**功能描述**：实时监测录音流，当检测到用户设定的唤醒词（姓名）时，触发提醒并弹出系统通知。

**技术实现**：
- 使用Sherpa-ONNX的KWS（Keyword Spotting）模型（通过JNI调用）
- 实时处理音频流，延迟<500ms
- 检测到唤醒词后触发回调函数
- Java 实现，跨平台兼容（通过 JNI 调用 C++ 库）

**唤醒词检测流程**：
```
1. 录音线程持续采集音频数据
2. 音频数据推送到唤醒词检测器
3. Sherpa-ONNX模型实时分析
4. 检测到唤醒词 → 触发事件
5. 发送系统通知 + 震动
```

**通知内容**：
- 标题："📢 老师点名啦！"
- 内容："检测到您的名字被叫到"
- 动作：点击跳转到应用

#### 3.2.4 功能4：安静检测（VAD）
**功能描述**：使用Sherpa-ONNX的VAD（Voice Activity Detection）模型检测课堂语音活动，当安静时间超过设定阈值（默认5秒可调）时，提醒用户老师可能准备提问。

**技术实现**：
- 使用Sherpa-ONNX的VAD模型（基于Silero，通过JNI调用）
- VAD模型基于深度学习，能准确区分"人类语音"和"环境噪音"
- 设置安静阈值（如5秒）
- 记录无语音活动持续时间
- 超时后触发提醒
- **Java 实现，通过 JNI 调用 C++ 库**

}

**用户配置**：
- 安静阈值：可调节（3-30秒，默认5秒）
- 开关控制：用户可关闭此功能

**VAD优势**：
- 比RMS音量检测更智能
- 能过滤翻书声、椅子移动等环境噪音
- 准确识别"是否有人在说话"
- 基于深度学习，准确率高

#### 3.2.5 功能5：语音识别
**功能描述**：当唤醒词检测或安静时间提醒触发后（默认唤醒词），自动获取触发时间点前N分钟（默认4分钟）的录音，使用Sherpa-ONNX语音识别模型转为文字。

**技术实现**：
- 从环形缓冲区提取指定时长的音频
- 使用Sherpa-ONNX的ASR（Automatic Speech Recognition）模型（通过JNI调用）
- 支持流式识别和批量识别
- 识别结果实时返回
- Java 实现，跨平台兼容（通过 JNI 调用 C++ 库）

**音频提取策略**：
```
1. 记录触发时间点：T_trigger
2. 计算目标时间范围：[T_trigger - 4min, T_trigger]
3. 从环形缓冲区提取该时间段的音频数据
4. 保存为临时WAV文件
5. 调用Sherpa-ONNX进行识别
6. 返回识别文字
```

**模型选择**：
- 推荐模型：`sherpa-onnx-zh-wenetspeech-zh`（中文识别）
- 特点：高精度，适合课堂环境
- 实时性：支持流式识别，延迟低

#### 3.2.6 功能6：AI问答
**功能描述**：将识别的文字内容按照模板发送给大模型，获取条理清晰、准确完整的回答，并直接显示在应用主界面中。

**技术实现**：
- 使用 LangChain4j 框架进行大模型调用
- 模型选择：支持多种模型（OpenAI GPT-4、百度千帆 ERNIE-Bot-turbo、阿里通义千问等）
- API Key：用户通过设置界面输入对应平台的 API Key / Access Token
- Prompt模板：固定模板 + 动态填充
- **使用 LangChain4j 统一 API，易于切换不同模型**

#### 4.2.1 环形缓冲区（Circular Buffer）- 线程安全
**目的**：高效管理音频数据，支持按时间范围提取

**实现思路（Java 原子操作）**：
```java
package com.classroomassistant.audio;

import java.util.concurrent.atomic.AtomicInteger;
import java.util.Arrays;

public class CircularBuffer {
    private final byte[] buffer;
    private final int bufferSize;
    private final AtomicInteger writeIndex = new AtomicInteger(0);
    private final AtomicInteger readIndex = new AtomicInteger(0);
    private final AtomicInteger dataSize = new AtomicInteger(0);

    public CircularBuffer(int size) {
        this.bufferSize = size;
        this.buffer = new byte[size];
    }

    /**
     * 写入音频数据（原子操作，线程安全）
     * @param data 音频数据
     */
    public void write(byte[] data) {
        for (byte b : data) {
            buffer[writeIndex.get()] = b;
            int newIndex = (writeIndex.get() + 1) % bufferSize;
            writeIndex.set(newIndex);

            if (dataSize.get() < bufferSize) {
                dataSize.incrementAndGet();
            } else {
                // 覆盖旧数据，更新读指针
                int newReadIndex = (readIndex.get() + 1) % bufferSize;
                readIndex.set(newReadIndex);
            }
        }
    }

    /**
     * 读取指定时长的音频数据（从当前时间点倒推）
     * @param seconds 秒数
     * @return 音频数据
     */
    public byte[] readBefore(int seconds) {
        int bytesToRead = seconds * 16000 * 2; // 16kHz, 16-bit
        byte[] result = new byte[bytesToRead];

        int startIndex = (writeIndex.get() - bytesToRead + bufferSize) % bufferSize;

        for (int i = 0; i < bytesToRead; i++) {
            result[i] = buffer[(startIndex + i) % bufferSize];
        }

        return result;
    }
}

**2. Java唤醒词检测器封装（WakeWordDetector.java）**：
```java
package com.classroomassistant.speech;

import java.io.FileWriter;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Paths;
import java.util.List;
import java.util.concurrent.CopyOnWriteArrayList;

/**
 * 唤醒词检测器（Java封装）
 */
public class WakeWordDetector {
    private final WakeWordDetectorJNI jni;
    private long detectorHandle = 0;
    private String keyword = "张三";
    private boolean initialized = false;

    // 监听器列表
    private final List<WakeWordListener> listeners = new CopyOnWriteArrayList<>();

    public interface WakeWordListener {
        void onWakeWordDetected(String keyword);
    }

    public WakeWordDetector() {
        this.jni = new WakeWordDetectorJNI();
    }

    /**
     * 添加监听器
     * @param listener 监听器
     */
    public void addListener(WakeWordListener listener) {
        listeners.add(listener);
    }

    /**
     * 移除监听器
     * @param listener 监听器
     */
    public void removeListener(WakeWordListener listener) {
        listeners.remove(listener);
    }

    /**
     * 初始化唤醒词检测器
     * @param modelPath 模型路径
     * @param keyword 唤醒词
     */
    public void initialize(String modelPath, String keyword) {
        this.keyword = keyword;

        try {
            // 1. 生成关键词配置文件
            String keywordsFilePath = generateKeywordsFile(keyword);

            // 2. 调用JNI初始化
            String encoderPath = modelPath + "/encoder.onnx";
            String decoderPath = modelPath + "/decoder.onnx";
            String joinerPath = modelPath + "/joiner.onnx";
            String tokensPath = modelPath + "/tokens.txt";

            detectorHandle = jni.initialize(
                encoderPath,
                decoderPath,
                joinerPath,
                tokensPath,
                keywordsFilePath
            );

            if (detectorHandle != 0) {
                initialized = true;
                System.out.println("Wake word detector initialized successfully");
            } else {
                System.err.println("Failed to initialize wake word detector");
            }
        } catch (Exception e) {
            System.err.println("Error initializing wake word detector: " + e.getMessage());
        }
    }

    /**
     * 生成关键词配置文件
     * @param keyword 唤醒词
     * @return 配置文件路径
     */
    private String generateKeywordsFile(String keyword) throws IOException {
        String tempDir = System.getProperty("java.io.tmpdir");
        String filePath = tempDir + "/keywords.txt";

        try (FileWriter writer = new FileWriter(filePath)) {
            // 支持多个唤醒词变体（逗号分隔）
            String[] keywords = keyword.split(",");
            for (String kw : keywords) {
                String trimmed = kw.trim();
                if (!trimmed.isEmpty()) {
                    writer.write(trimmed + " 0.5\n");
                }
            }
        }

        return filePath;
    }

    /**
     * 检测唤醒词
     * @param audioData 音频数据
     * @return 是否检测到唤醒词
     */
    public boolean detect(float[] audioData) {
        if (!initialized || detectorHandle == 0) {
            return false;
        }

        // 调用JNI检测
        int result = jni.detect(detectorHandle, audioData, 16000);

        if (result == 1) {
            String detectedKeyword = jni.getDetectedKeyword(detectorHandle);
            if (detectedKeyword != null && !detectedKeyword.isEmpty()) {
                // 通知所有监听器
                for (WakeWordListener listener : listeners) {
                    listener.onWakeWordDetected(detectedKeyword);
                }
                return true;
            }
        }

        return false;
    }

    /**
     * 设置唤醒词
     * @param keyword 唤醒词
     */
    public void setKeyword(String keyword) {
        this.keyword = keyword;
    }

    /**
     * 释放资源
     */
    public void destroy() {
        if (detectorHandle != 0) {
            jni.destroy(detectorHandle);
            detectorHandle = 0;
            initialized = false;
        }
    }

     */

            jni.destroy(detectorHandle);
            detectorHandle = 0;
            initialized = false;

    /**
     * 下载模型（同步）
     * @param modelInfo 模型信息
     */
    public void downloadModel(ModelInfo modelInfo) {
        HttpURLConnection connection = null;
        InputStream inputStream = null;
        FileOutputStream outputStream = null;

        try {
            URL url = new URL(modelInfo.url);
            connection = (HttpURLConnection) url.openConnection();
            connection.connect();

            // 检查响应码
            int responseCode = connection.getResponseCode();
            if (responseCode != HttpURLConnection.HTTP_OK) {
                throw new IOException("Server returned HTTP " + responseCode);
            }

            // 获取文件大小
            int fileSize = connection.getContentLength();
            inputStream = connection.getInputStream();

            // 创建目标目录
            Path parentDir = Paths.get(modelInfo.savePath).getParent();
            Files.createDirectories(parentDir);

            // 创建输出流
            outputStream = new FileOutputStream(modelInfo.savePath);
            byte[] buffer = new byte[4096];
            int bytesRead;
            long totalBytesRead = 0;

            // 下载文件
            while ((bytesRead = inputStream.read(buffer)) != -1) {
                outputStream.write(buffer, 0, bytesRead);
                totalBytesRead += bytesRead;

                // 通知进度
                if (fileSize > 0) {
                    double progress = (double) totalBytesRead / fileSize * 100;
                    for (DownloadListener listener : listeners) {
                        listener.onDownloadProgress(modelInfo.name, progress);
                    }
                }
            }

            // 通知下载完成
            for (DownloadListener listener : listeners) {
                listener.onDownloadFinished(modelInfo.name);
            }

        } catch (Exception e) {
            String errorMsg = "Failed to download model: " + e.getMessage();
            System.err.println(errorMsg);

            // 通知下载失败
            for (DownloadListener listener : listeners) {
                listener.onDownloadFailed(modelInfo.name, errorMsg);
            }

            // 删除不完整的文件
            try {
                Files.deleteIfExists(Paths.get(modelInfo.savePath));
            } catch (IOException ignored) {
            }

        } finally {
            // 关闭资源
            if (inputStream != null) {
                try { inputStream.close(); } catch (IOException ignored) {}
            }
            if (outputStream != null) {
                try { outputStream.close(); } catch (IOException ignored) {}
            }
            if (connection != null) {
                connection.disconnect();
            }
        }
    }

void ModelDownloader::downloadModel(const ModelInfo& modelInfo) {
    QUrl url(modelInfo.url);
    QNetworkRequest request(url);

    QNetworkReply *reply = m_networkManager->get(request);

    connect(reply, &QNetworkReply::downloadProgress, this,
            [this, modelInfo](qint64 bytesReceived, qint64 bytesTotal) {
        if (bytesTotal > 0) {
            double progress = (double)bytesReceived / bytesTotal * 100;
            emit downloadProgress(modelInfo.name, progress);
        }
    });

    connect(reply, &QNetworkReply::finished, this, [this, reply, modelInfo]() {
        if (reply->error() == QNetworkReply::NoError) {
            // 保存文件
            QDir().mkpath(QFileInfo(modelInfo.savePath).absolutePath());
            QFile file(modelInfo.savePath);
            if (file.open(QIODevice::WriteOnly)) {
                file.write(reply->readAll());
                file.close();

                // 如果是压缩文件，自动解压
                if (modelInfo.savePath.endsWith(".tar.bz2") || modelInfo.savePath.endsWith(".tar.gz")) {
                    QString destPath = QFileInfo(modelInfo.savePath).absolutePath();
                    extractModel(modelInfo.savePath, destPath);
                }

                emit downloadFinished(modelInfo.name);
            } else {
                emit downloadFailed(modelInfo.name, "Failed to save file");
            }
        } else {
            emit downloadFailed(modelInfo.name, reply->errorString());
        }
        reply->deleteLater();
    });
}

package com.classroomassistant.utils;

import java.util.concurrent.CopyOnWriteArrayList;

/**
 * 错误处理器（Java封装）
 */
public class ErrorHandler {
    private static ErrorHandler instance;
    private final List<ErrorListener> listeners = new CopyOnWriteArrayList<>();

    public enum ErrorCode {
        SUCCESS(0),
        MODEL_LOAD_FAILED(1001),
        MODEL_FILE_NOT_FOUND(1002),
        NETWORK_ERROR(2001),
        INVALID_API_KEY(2002),
        AUDIO_PERMISSION_DENIED(3001),
        AUDIO_DEVICE_NOT_AVAILABLE(3002),
        RECOGNITION_FAILED(4001);

        private final int code;

        ErrorCode(int code) {
            this.code = code;
        }

        public int getCode() {
            return code;
        }
    }

    public interface ErrorListener {
        void onErrorOccurred(String errorMessage);
    }

    private ErrorHandler() {
    }

    public static ErrorHandler getInstance() {
        if (instance == null) {
            instance = new ErrorHandler();
        }
        return instance;
    }

    /**
     * 添加监听器
     */
    public void addListener(ErrorListener listener) {
        listeners.add(listener);
    }

    /**
     * 移除监听器
     */
    public void removeListener(ErrorListener listener) {
        listeners.remove(listener);
    }

    /**
     * 处理错误
     * @param code 错误码
     * @param message 错误消息
     */
    public void handleError(ErrorCode code, String message) {
        String errorMsg;

        switch (code) {
            case MODEL_LOAD_FAILED:
                errorMsg = "模型加载失败: " + message;
                break;
            case MODEL_FILE_NOT_FOUND:
                errorMsg = "模型文件不存在，请先下载模型";
                break;
            case NETWORK_ERROR:
                errorMsg = "网络错误: " + message;
                break;
            case INVALID_API_KEY:
                errorMsg = "API密钥无效，请检查设置";
                break;
            case AUDIO_PERMISSION_DENIED:
                errorMsg = "录音权限被拒绝，请在设置中开启权限";
                break;
            case AUDIO_DEVICE_NOT_AVAILABLE:
                errorMsg = "音频设备不可用";
                break;
            case RECOGNITION_FAILED:
                errorMsg = "识别失败: " + message;
                break;
            default:
                errorMsg = "未知错误: " + message;
        }

        System.err.println("Error: " + code.getCode() + " " + errorMsg);

        // 通知所有监听器
        for (ErrorListener listener : listeners) {
            listener.onErrorOccurred(errorMsg);
        }
    }

#### 5.2.1 步骤1：实现音频录制模块（Java）
**文件**：`com/classroomassistant/audio/AudioRecorder.java`

```java
package com.classroomassistant.audio;

import javax.sound.sampled.*;
import java.io.ByteArrayOutputStream;
import java.util.ArrayList;
import java.util.List;

public class AudioRecorder {

    private boolean isRecording = false;
    private TargetDataLine microphone;
    private CircularBuffer circularBuffer;
    private static final int SAMPLE_RATE = 16000;
    private static final int BITS_PER_SAMPLE = 16;
    private static final int CHANNELS = 1;
    private static final int BUFFER_DURATION = 300; // 5 minutes (in seconds)

    public interface AudioListener {
        void onAudioReady(byte[] data);
        void onError(String error);
    }

    private List<AudioListener> listeners = new ArrayList<>();

    public AudioRecorder() {
        int bufferSize = SAMPLE_RATE * BITS_PER_SAMPLE / 8 * CHANNELS * BUFFER_DURATION;
        circularBuffer = new CircularBuffer(bufferSize);
    }

    public void addListener(AudioListener listener) {
        listeners.add(listener);
    }

    public void removeListener(AudioListener listener) {
        listeners.remove(listener);
    }

    public void startRecording() {
        try {
            AudioFormat format = new AudioFormat(
                SAMPLE_RATE,
                BITS_PER_SAMPLE,
                CHANNELS,
                true, // signed
                false // little endian
            );

            DataLine.Info info = new DataLine.Info(TargetDataLine.class, format);
            microphone = (TargetDataLine) AudioSystem.getLine(info);
            microphone.open(format);
            microphone.start();

            isRecording = true;

            // 启动录音线程
            Thread recordingThread = new Thread(this::recordAudio);
            recordingThread.start();

        } catch (LineUnavailableException e) {
            notifyError("无法访问麦克风: " + e.getMessage());
        }
    }

    private void recordAudio() {
        byte[] buffer = new byte[4096];
        ByteArrayOutputStream outputStream = new ByteArrayOutputStream();

        while (isRecording) {
            int bytesRead = microphone.read(buffer, 0, buffer.length);
            if (bytesRead > 0) {
                // 写入环形缓冲区
                circularBuffer.write(buffer, 0, bytesRead);
            }
        }
    }

    public void stopRecording() {
        isRecording = false;
        if (microphone != null) {
            microphone.stop();
            microphone.close();
        }
    }

    public boolean isRecording() {
        return isRecording;
    }

    public byte[] getAudioBefore(int seconds) {
        int size = SAMPLE_RATE * BITS_PER_SAMPLE / 8 * CHANNELS * seconds;
        return circularBuffer.readBefore(size);
    }

    private void notifyAudioReady(byte[] data) {
        for (AudioListener listener : listeners) {

**音频录制实现说明**：

音频录制功能的JNI层C++实现负责：
1. 初始化环形缓冲区
2. 使用QAudioSource进行音频采集
3. 将音频数据写入环形缓冲区
4. 通过信号通知Java层音频数据可用

Java侧通过调用AudioRecorder类的方法来控制录音，无需关心底层实现。

void AudioRecorder::startRecording() {
    // 配置音频格式
    QAudioFormat format;
    format.setSampleRate(SAMPLE_RATE);
    format.setChannelCount(1);
    format.setSampleFormat(QAudioFormat::Int16);
    format.setSampleType(QAudioFormat::SignedInt);

    // 获取默认音频输入设备
    QAudioDevice audioDevice = QMediaDevices::defaultAudioInput();
    if (audioDevice.isNull()) {
        emit errorOccurred("No audio input device available");
        return;
    }

    // 创建音频源
    m_audioSource = new QAudioSource(audioDevice, format, this);

    // 开始录音
    m_audioDevice = m_audioSource->start();

    // 连接数据就绪信号
    connect(m_audioSource, &QAudioSource::readyRead, this, &AudioRecorder::onAudioReady);

    m_isRecording = true;
    emit isRecordingChanged();
}

void AudioRecorder::stopRecording() {
    if (m_audioSource) {
        m_audioSource->stop();
        m_audioDevice = nullptr;
        m_isRecording = false;
        emit isRecordingChanged();
    }
}

void AudioRecorder::onAudioReady() {
    if (!m_audioDevice) {
        return;
    }

    // 读取音频数据
    QByteArray audioData = m_audioDevice->readAll();

    // 写入环形缓冲区
    m_circularBuffer->write(audioData.constData(), audioData.size());

    // 转换为Float数组（Sherpa-ONNX需要float格式）
    const qint16 *data = reinterpret_cast<const qint16*>(audioData.constData());
    int sampleCount = audioData.size() / sizeof(qint16);
    std::vector<float> floatData(sampleCount);

    for (int i = 0; i < sampleCount; ++i) {
        floatData[i] = static_cast<float>(data[i]) / 32768.0f;
    }

    // 发射信号（供检测线程使用）
    emit audioReady(floatData);
}

QByteArray AudioRecorder::getAudioBefore(int seconds) {
    int bytesToRead = seconds * SAMPLE_RATE * 2; // 16-bit
    std::vector<char> rawData = m_circularBuffer->readBefore(seconds);

    return QByteArray(rawData.data(), rawData.size());
}

### 7.3 技术风险与应对

#### 7.3.1 唤醒词检测误报
**风险描述**：环境噪音导致误报唤醒

**应对措施**：
- 设置合理的检测阈值（0.5）
- 用户可调节灵敏度
- 使用VAD模型过滤噪音

#### 7.3.2 语音识别准确率
**风险描述**：课堂环境噪音大，识别准确率低

**应对措施**：
- 提供识别结果预览，允许用户手动修改
- 使用更大的模型（权衡模型大小和准确率）
- 后续支持云端识别API（备选方案）

#### 7.3.3 设备性能不足
**风险描述**：低端设备运行卡顿，电池消耗快

**应对措施**：
- 提供性能设置选项（降低采样率、检测频率）
- 使用ONNX的量化模型（INT8）
- 后台运行显示通知，避免被系统杀死
- 优化内存和CPU使用（原子操作）

#### 7.3.4 网络不稳定
**风险描述**：AI API调用失败

**应对措施**：
- 本地缓存识别结果，稍后重试
- 提供离线模式（仅语音识别，无AI问答）
- 网络检测，网络差时提示用户
- 支持多个AI服务商（降低单点故障）
- 用户可配置API密钥，灵活切换

#### 7.3.5 模型下载失败
**风险描述**：网络问题导致模型下载失败

**应对措施**：
- 支持断点续传
- 提供重试机制
- 显示下载进度
- 下载完成后验证文件完整性

    @Test
    void testWriteAndRead() {
        // 写入数据
        byte[] testData = "Hello, World!".getBytes();
        buffer.write(testData);

        // 读取数据
        byte[] readData = buffer.readBefore(1);
        assertArrayEquals(testData, readData);

    @Test
    void testThreadSafety() throws InterruptedException {
        final int durationMs = 1000;
        final AtomicBoolean running = new AtomicBoolean(true);
        final AtomicInteger writeCount = new AtomicInteger(0);
        final AtomicInteger readCount = new AtomicInteger(0);

        // 写入线程
        Thread writer = new Thread(() -> {
            byte[] data = "Test data".getBytes();
            while (running.get()) {
                buffer.write(data);
                writeCount.incrementAndGet();
            }
        });

        // 读取线程
        Thread reader = new Thread(() -> {
            while (running.get()) {
                buffer.readBefore(1);
                readCount.incrementAndGet();
            }
        });

        // 启动线程
        writer.start();
        reader.start();

        // 运行1秒
        Thread.sleep(durationMs);
        running.set(false);

        // 等待线程结束
        writer.join();
        reader.join();

        // 验证无崩溃
        assertTrue(writeCount.get() > 0);
        assertTrue(readCount.get() > 0);
    }

    @Test
    void testOverwrite() {
        CircularBuffer smallBuffer = new CircularBuffer(100);

        // 写入超过缓冲区大小的数据
        byte[] testData = new byte[200];
        Arrays.fill(testData, (byte) 'A');

        // 读取最后50字节

        assertEquals(50, readData.length);
        assertEquals((byte) 'A', readData[0]);
    }

    @Test

        assertEquals(0, buffer.getDataSize());

        byte[] data = "Test".getBytes();
        buffer.write(data);
