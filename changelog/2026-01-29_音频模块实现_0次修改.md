## 1. 需求描述

实现桌面端音频采集与环形缓冲区，支持回溯读取，为后续 KWS/VAD/ASR 提供输入。

## 2. 实现思路

### 2.1 技术方案

- 使用 Java Sound API 采集 TargetDataLine PCM 数据。
- 采用固定容量环形缓冲区存储音频字节，支持覆盖写入与回溯读取。

### 2.2 架构设计

- audio 模块提供 `AudioRecorder` 接口与 `AudioRecorderDesktop` 实现。
- `CircularBuffer` 负责音频数据缓存。

### 2.3 技术选型

- Java Sound API（JDK 内置，无额外依赖）。
