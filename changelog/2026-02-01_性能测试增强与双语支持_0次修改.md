# 变更日志 - 性能测试增强与双语支持

**日期**: 2026-02-01

**版本**: 0.5.1

## 概述

本次迭代完成了性能测试套件的扩展，新增了 AI 模块和语音模块的性能测试，并完善了中英双语国际化支持。

---

## 一、性能测试框架概述

### 1. 双语消息系统
项目采用 MessageFactory 工厂模式实现中英双语消息切换：

```java
// MessageFactory 根据 app.language 配置加载对应消息实现
public static PerformanceTestMessages getMessages() {
    String lang = ConfigManager.getInstance().getString("app.language", "zh");
    return "en".equals(lang) ? new EnglishMessages() : new ChineseMessages();
}
```

### 2. 配置方式
在 `src/main/resources/config/application.properties` 中设置：
```properties
# 语言设置 / Language setting
# zh = 中文, en = English
app.language=zh
```

## 三、Mock 实现详情

### 1. MockLLMClient
模拟 LangChain4j LLM 客户端，支持：
- 同步调用 `generate()`
- 异步回调 `generateAsync()`
- 可配置模拟延迟

### 2. MockWakeWordDetector
模拟唤醒词检测器，基于 RMS 阈值判断：
- 实现 `WakeWordDetector` 接口
- 支持多关键词初始化
- 监听器回调机制

### 3. MockSilenceDetector
模拟静音检测 (VAD)：
- 实现 `SilenceDetector` 接口
- 基于 RMS 阈值判断语音/静音
- 静音超时回调

### 4. MockSpeechRecognizer
模拟语音识别引擎：
- 实现 `SpeechRecognizer` 接口
- 支持同步/异步识别
- 可配置处理延迟

## 四、性能测试执行

### 运行所有性能测试
```bash
mvn test -Dgroups=performance
```

### 仅运行核心模块性能测试
```bash
mvn test -Dtest=CorePerformanceTest
```

### 仅运行 AI 模块性能测试
```bash
mvn test -Dtest=AIPerformanceTest
```

### 仅运行语音模块性能测试
```bash
mvn test -Dtest=SpeechPerformanceTest
```
