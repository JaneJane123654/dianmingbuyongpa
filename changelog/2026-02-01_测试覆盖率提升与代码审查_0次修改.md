## 二、新增测试类

### 1. WavWriterTest
- 验证 WAV 文件头生成（RIFF、WAVE、fmt、data 块）
- 测试 PCM 数据正确写入
- 测试采样率、声道数等参数写入
- 测试空 PCM 和嵌套目录处理
- 测试 null 参数校验

### 2. AudioFormatSpecTest
- 验证构造函数参数设置
- 测试 bytesPerSecond 计算（单/双声道，8/16bit）
- 测试 frameBytes 计算
- 测试 toAudioFormat 转换
- 测试 equals/hashCode/toString

### 3. ModelDownloadManagerTest
- 测试构造函数 null 参数校验
- 测试缺失模型检测逻辑
- 验证 JavaFX 属性初始值
- 测试模型描述符生成

### 4. ModelDescriptorTest
- 验证值对象行为
- 测试可选参数为 null 的情况
- 测试 equals/hashCode 各种场景

### 5. ModelDownloaderTest
- 测试 cancel/reset 方法
- 测试 null 参数校验
- 测试无效 URL 错误处理
