# Session 模块

## 模块描述

会话编排模块负责串联录音、语音检测与 AI 问答的业务流程，统一管理状态与 UI 绑定。

## 类/接口说明

| 类/接口 | 说明 |
| --- | --- |
| `ClassSessionManager` | 会话编排器，管理状态与流程 |
| `SessionState` | 会话状态枚举 |

## 注意事项

1. UI 更新需使用 JavaFX 线程（`Platform.runLater()`）。
2. 只有 `ClassSessionManager` 负责流程编排，避免模块直连。
